使用说明

如何生成密钥

  1. 选择密钥类型（标准、Microsoft、简单或自定义格式）
  2. 设置生成数量（1-100个）
  3. 点击"生成密钥"按钮
  4. 点击单个密钥或使用"复制全部"按钮复制

如何部署

  1. 点击上方的一键部署按钮
  2. 登录您的 Cloudflare/Vercel/Netlify 账户
  3. 按照提示完成部署
  4. 部署后即可通过 API 调用生成密钥

API 使用

// GET 请求
fetch('/api/generate?count=5&type=microsoft')
  .then(res => res.json())
  .then(keys => console.log(keys))

// POST 请求
fetch('/api/generate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ count: 3, type: 'standard' })
})
